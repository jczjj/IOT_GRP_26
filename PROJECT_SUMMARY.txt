   ─────────────────────────────────────────────────────────────────────────

   √ Image available indicators

4. [LOCATION SERVICES]
   ─────────────────────────────────────────────────────────────────────────

   √ Location update command simulation
   √ Visual feedback in 3D view
   √ Distance calculation from RSSI values
   √ Path loss exponent formula implementation
   √ Coordinate display (meters)

5. [REST API ENDPOINTS]
   ─────────────────────────────────────────────────────────────────────────
   √ GET  /api/devices                  → List all devices
   √ GET  /api/stationary-nodes         → List all anchor nodes
   √ GET  /api/device/{id}              → Get specific device
   √ POST /api/locate/{id}              → Trigger location update
   √ POST /api/request-image/{id}       → Request image capture
   √ GET  /api/device/{id}/image        → Get image metadata
   √ GET  /                             → Dashboard view
   √ GET  /view-image/{id}              → Image viewer
   √ GET  /health                       → Health check

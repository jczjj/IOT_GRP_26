# Dummy Image Placeholder

This directory should contain captured images from patient devices.

For testing purposes, you can:
1. Add any .jpg images here
2. Name them using the pattern: {device_id}_latest.jpg
3. Or run the generate_dummy_images.py script

Examples:
- device_001_latest.jpg
- device_002_latest.jpg
- device_004_latest.jpg

In production, images will be automatically saved here when devices transmit via Wi-Fi hopping.

Image format recommendations:
- Format: JPEG
- Resolution: 640x480 or 1280x720
- Max size: 5MB
